HTML

HTML stand for HyperText Markup Language, is the standard markup language used for creating web pages and web applications. It provides a structure for organizing and presenting content on including text, images, links, forms, and multimedia elements.

CSS

CSS stand for Cascading Style Sheets, is a style sheet language used to control the presentation and layout of web pages written in HTML and XHTML.

JavaScript

JavaScript is a type of computer code used to make web pages interactive. It can change things on a page, respond to what users do, and communicate with other computer programs.
